A modern, responsive, and high-performance personal portfolio website built with HTML5, CSS3, Bootstrap 5, and JavaScript. Designed for front-end developers, graphic designers, and UI/UX designers to present their projects, skills, education, and professional experience beautifully.

---

## 🌟 Key Features

*   **Responsive Design:** Fully mobile-first layout utilizing Bootstrap 5 components.
*   **Video Background:** Dynamic and engaging video hero section with high optimization.
*   **Interactive Navigation:** Smooth-scrolling navbar with active-state tracking using jQuery.
*   **Aesthetic UI/UX:** Clean hover transitions, cards, and professional color scheme.
*   **Interactive Contact Form:** Built-in form linked to FormSubmit for instant email notifications.
*   **Embedded Location:** Integrated Google Maps tab to showcase location.
*   **Highly Optimized Assets:** 99.4% file-size reduction on images for instant global load times.

## 🚀 Getting Started

### Prerequisites
To run this project locally, all you need is a web browser. No complex package installations are required!

### Running Locally
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Shivaniroy09/Portfolio.git
   ```
2. Navigate into the project folder:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your web browser, or serve it using a lightweight HTTP server (recommended):
   ```bash
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.
